## Reporting a Vulnerability

Security is a high priority for CampusConnect, particularly regarding Supabase keys, RLS (Row Level Security) policies, and user data.

If you discover a vulnerability, **do NOT open a public issue.** Instead, please responsibly disclose it by emailing the project maintainer directly at:

**[INSERT MAINTAINER EMAIL HERE]**

Please include the following in your report:
- A description of the vulnerability
- Steps to reproduce it
- Any potential impact on user data or the platform

We will acknowledge your report as quickly as possible and work with you to resolve the issue safely before public disclosure.
